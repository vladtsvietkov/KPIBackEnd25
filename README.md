## Інструкція із запуску (docker-compose)
Щоб розгорнути проект локально, виконайте наступні кроки:  
1. **Завантажте вихідний код:**  
```bash
git clone https://github.com/vladtsvietkov/KPIBackEnd25.git
cd KPIBackEnd25  
```
2. **Збірка та запуск контейнерів:**    
Перебуваючи у головній директорії проєкту, виконайте команду для ініціалізації:   
```bash
docker-compose up --build 
```
3. **Доступ до сервісу**  
Після успішного запуску сервер прийматиме запити за посиланням http://localhost:8080  
4. **Завершення роботи**  
Щоб зупинити сервіс та видалити створені контейнери, скористайтеся командою:  
```bash
docker-compose down
```
## Діагностика працездатності  
Ви можете перевірити статус сервера, звернувшись до спеціального маршруту перевірки стану.
  * Endpoint - /healtcheck
  * HTTP метод - GET  

Приклад виклику через термінал:
```bash
curl http://localhost:8080/healthcheck
```  
Формат відповіді у разі успіху:  
```json
{
  "status": "OK",
  "message": "Backend is functional"
}
```
